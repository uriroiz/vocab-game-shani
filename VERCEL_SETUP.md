# 🔧 התקנת Vercel GitHub App

## מה כבר בוצע:
✅ דף ההתקנה נפתח בדפדפן

## מה לעשות עכשיו:

### שלב 1: התקנת Vercel GitHub App (30 שניות)

1. בדפדפן שנפתח, תראה דף התקנה של Vercel
2. בחר **"Only select repositories"**
3. בחר את **`vocab-game-maayan`**
4. לחץ **"Install"**
5. אשר את ההרשאות

### שלב 2: פרסום אוטומטי

לאחר ההתקנה, Vercel יזהה את ה-repository החדש ויפרסם אותו אוטומטית!

**זה יקרה תוך 2-3 דקות אחרי ההתקנה.**

---

## 🔄 אם לא קרה אוטומטי:

אם אחרי 5 דקות המשחק לא פורסם, עשה:

1. היכנס ל-[vercel.com/new](https://vercel.com/new)
2. לחץ על **"Import Git Repository"**
3. בחר **`uriroiz/vocab-game-maayan`**
4. לחץ **"Deploy"**

---

## ✅ אחרי הפרסום:

תקבל קישור כמו:
```
https://vocab-game-maayan.vercel.app
```

**זהו! המשחק חי ואונליין!** 🎉
